# SFML-build-test
Тестовый проект для отладки скачанных пакетов стека SFML3

#
### Компилятор [GNU g++ 14.2.0](https://github.com/brechtsanders/winlibs_mingw/releases?page=4)

### Графическая библиотека [SFML 3.0.2](https://www.sfml-dev.org/download/sfml/3.0.2/)

### Система автоматизации сборки [CMake 3.28+](https://cmake.org/download/)

#
##### Компиляция
Подготовка директории сборки CMake. Предлагаю договориться использовать `cmake-build`
для промежуточных файлов CMake (эта директория уже добавлена в [`.gitignore`](./.gitignore))
и `.build` для итоговой релизной сборки приложения (эта директория уже зашита в
[`CMakeLists.txt`](./CMakeLists.txt))

Команды для конфигурации CMake
1. `cmake -S . -B cmake-build` (настройка CMake, один раз после клонирования репозитория)
2. `cmake --build cmake-build` (вызов компиляции, каждый раз)

Вообще, можно просто открыть через утилиту CMake в VSCode или CLion и не париться

##### Перед запуском скомпилированного бинарника
В директорию вместе с ним необходимо положить `.dll` файлы SFML
(скопировать можно отсюда: `SFML-3.0.2\bin`)
